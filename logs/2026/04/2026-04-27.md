# 2026-04-27

## Project
RealityCheck Live — Ship to Prod Hackathon

## What was built
- Full-stack voice-first startup idea validation agent
- Vapi voice intake with demo simulation mode (waveform animation, pulsing rings, mic icon)
- 5-lane parallel research pipeline via TinyFish (competitors, substitutes, pricing, pain, why-now)
- Redis run state store with 24h TTL, event log, and evidence lists
- Groq (Llama 3.1) LLM synthesis into structured Market Atlas with score 0–100
- Next.js 15 API routes: POST /api/runs, GET /api/runs/:id, POST /api/vapi/tools
- Vapi dual-format tool call normalization (toolCallList + toolCalls.function)
- Live polling dashboard with EventFeed, EvidenceCard, SponsorTrace pipeline
- Demo mode with DEMO_RUN fixture and simulated running run
- Comprehensive UI visual upgrade:
  - Animated background orbs (cyan/fuchsia/lime/violet) in layout
  - Scan-line animation across viewport
  - Custom keyframe library in globals.css (shimmer, glow-pulse, float, wave, ring-expand, score-reveal)
  - SVG circular score gauge with gradient arc, glow filter, and tick marks in MarketAtlas
  - Audio waveform bars during active Vapi call
  - Shimmer skeleton loading states
  - Sponsor pipeline trace with live ping dots
