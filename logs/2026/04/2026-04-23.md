# 2026-04-23

## Project
woop — health & fitness dashboard

## What was built
- Ring stack component with animated progress rings (activity, exercise, stand)
- HRV (Heart Rate Variability) chart with trend visualization
- Contribution heatmap for workout history
- Sleep coach feature: bedtime target display on /today and /sleep pages
- Stress monitor using HR coefficient-of-variation + mood fallback
- Performance assessment page with full PDF export support

## Tech
Next.js · TypeScript · Tailwind CSS · Recharts · jsPDF

## Notes

- HRV chart falls back gracefully when data is sparse
- Turbopack root pinned to project dir to fix stale lockfile issue
